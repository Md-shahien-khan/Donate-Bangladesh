Important FAQ

What is the Document Object Model(DOM)?

Dom Stands for Document Object Model. Which is a programming interface that represents a web page's structure as tree of nodes and objects.

How do you select an element from the DOM

Using getElementByid, getElementByClassName, getElementByTagName, querySelector, querySelectorAll etc.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation refers to the process of suing event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated. It allows us to attach a single event listener for elements that exist now or in the future.

How do you manipulate an element's attributes and styles using DOM?

The classList object only provide methods change the class of an element. If you want to change other attributes like id, href, or src, you can use that setAttribute() method.
